\begin{problem}{Квадратное уравнение}{quadratic.in}{quadratic.out}{1 секунда}

\def\bbZ{\mathbb{Z}}%
Детей в школе учат решать квадратные уравнения, т.е. уравнения вида
$$ax^2+bx+c=0,$$ где $a$, $b$ и $c$ некоторые заданные действительные
числа, а $x$~--- действительное число, которое необходимо найти.

В этой задаче вам потребуется решить квадратное уравнение для
многочленов с коэффициентами из $\bbZ/2\bbZ$. Напомним, что в $\bbZ/2\bbZ$
есть только два числа: $0$ и $1$, и все операции в этом поле производятся
по модулю 2.

Даны многочлены $a(t)$, $b(t)$ и $c(t)$, найдите такой полином $x(t)$ что
$$a(t)x^2(t)+b(t)x(t)+c(t)=0,$$ где равенство понимается как равенство
многочленов. Напомним, что многочлены равны тогда и только тогда, когда
равны их коэффициенты при соответствующих степенях $t$.

\InputFile

Входной файл содержит многочлены $a(t)$, $b(t)$ и $c(t)$, которые задаются
их степенями, за которыми следуют коэффициенты, начиная со старшего.
Нулевые многочлены в данной задаче имеют степень $-1$. Будем считать, что
степени всех многочленов не превосходят $127$.

\OutputFile

Если есть хотя бы одно решение уравнения, выведите любое из них в таком же
формате. Старший коэффициент найденного многочлена не должен быть нулевым.
Степень полинома не должна превышать $512$.

В противном случае напечатайте ``\texttt{no solution}''
в первой строке выходного файла.

\Example

\begin{example}
\exmp{
0  1
2  1 1 0
3  1 0 0 0 
}{
1  1 0
}%
\exmp{
0  1
1  1 1
0  1
}{
no solution
}%
\exmp{
-1
-1
-1
}{
-1
}%
\end{example}
\medskip

В первом примере многочлен $x(t) = t$ является решением уравнения $x(t)^2+(t^2+t)x(t)+t^3=0$.

Во втором примере уравнение $x(t)^2+(t+1)x(t)+1=0$ не имеет решения.

В третьем примере уравнение имеет вид $0 = 0$ и любой многочлен является его решением.

\end{problem}
