\begin{problem}{A + B}{aplusb.in}{aplusb.out}{2 секунды}{64 мегабайта}

Цепной дробью называется выражение вида
\begin {equation*}
a_0 + \frac {1} {a_1 + \frac {1} {a_2 + \frac {1}
 {\ldots + \frac {1} {a_n}}}},
\end {equation*}
где все $a_k$ целые, $a_k > 0$ при $k > 0$ и $a_n > 1$, если $n > 0$.

Эти правила позволяют каждому вещественному числу $x$ единственным образом
сопоставить цепную дробь так, чтобы выполнялось равенство
\begin {equation*}
x = a_0 + \frac {1} {a_1 + \frac {1} {a_2 + \frac {1}
 {\ldots + \frac {1} {a_n}}}}.
\end {equation*}

Например, цепная дробь для числа $\frac {7} {18}$ выглядит как
$0 + \frac {1} {2 + \frac {1} {1 + \frac {1} {1 + \frac {1} {3}}}}$.

Цепную дробь можно записывать как $[a_0; a_1, a_2, \ldots, a_n]$;
в этой записи $\frac {7} {18} = [0; 2, 1, 1, 3]$.

Не все числа имеют конечную цепную дробь; так, для числа $\sqrt {2}$
цепная дробь имеет вид $[1; 2, 2, 2, 2, \ldots]$. Эта цепная дробь ---
периодическая; можно записывать е\"е как $[1; (2)]$. Индекс элемента,
с которого начинается период, выбирается минимально возможным положительным
числом, как и его длина; к примеру, запись $[1; 2, (2, 2)]$ не считается
корректной.

Даны две периодические цепные дроби. Требуется вывести их сумму, если она
представима в виде периодической цепной дроби, или слово ``\t{Impossible}''
в противном случае.

\InputFile

Входной файл содержит две строки. В каждой строке содержится запись цепной
дроби в формате $[a_0; a_1, \ldots, a_k, (a_{k + 1}, \ldots, a_n)]$. Эта
запись означает, что цепная дробь имеет предпериод $a_1, \ldots, a_k$ и
период $(a_{k + 1}, \ldots, a_n)$; таким образом, $a_{n + 1} = a_{k + 1}$,
$a_{n + 2} = a_{k + 2}$, \ldots. Для обеих данных цепных дробей верно, что
$0 \leq k < n \leq 7$ и $1 \leq a_i \leq 10$. Строки не содержат пробелов.

\OutputFile

Если сумма двух заданных цепных дробей не является периодической цепной
дробью, выведите в выходной файл строку ``\t{Impossible}''. В противном
случае выведите цепную дробь в таком же формате. Выведенная строка не должна
содержать пробелов. Гарантируется, что в этой строке не более $1000$ чисел.

\Examples

\begin{example}%
\exmp{
[1;(2)]
[2;(1,4)]
}{
[4;(4,8)]
}%
\end{example}

\end{problem}
