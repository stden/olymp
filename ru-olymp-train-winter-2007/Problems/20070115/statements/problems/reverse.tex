\begin{problem}{Развернутые числа}{reverse.in}{reverse.out}{2 секунды}

Определим \emph{разворот} $R(x)$ числа $x$ как число, десятичная запись которого 
представляет собой десятичную запись числа $x$, цифры которой записаны
в обратном порядке.

Например, $R(123)=321$ или $R(1200)=21$.

Вам дано несколько целых чисел. Для каждого из них требуется определить,
может ли оно быть равно $x + R(x)$ для некоторого $x$.

\InputFile

Входной файл содержит не более $10\,001$ строки. Каждая строка содержит
положительное целое число, содержащее не более $10\,000$ цифр. 
Последняя строка входного файла содержит 0, это число обрабатывать не
следует. Размер входного файла не превышает 200 килобайт.

\OutputFile

Выведите в выходной файл по одной строке для каждого числа. Если
число может быть представлено в виде $x + R(x)$ для некоторого $x$,
выведите ``\t{YES}'', иначе выведите ``\t{NO}''.

\end{problem}
