\begin{problem}{Трибитный компьютер}{bit.in}{bit.out}{2 секунды}

Уч\"еные королевства Байтландии решили создать новый тип компьютера ---
трибитный компьютер (ТБК). По их прогнозам выходит, что эта машина
сможет помочь в решении многих сложных и ещ\"е не реш\"енных задач.
Однако, сначала им предстоит справиться с некоторыми техническими
сложностями. Ваша задача --- помочь уч\"еным решить одну из возникших
проблем.

Уч\"еных сейчас интересует то, как будет инициализироваться память нового
компьютера. Тестовая сборка ТБК имеет $n$ битов памяти, пронумерованных от $1$
до $n$. Каждый бит может принимать одно из тр\"ех значений \t{a}, \t{b}, \t{c},
или же пребывать в неинициализированном состоянии. ТБК поддерживает следующие
две операции для инициализации битов своей памяти:
\begin {itemize}
 \item Записать в два неинициализированных подряд идущих бита, то есть в биты
  с номерами $i$ и $i + 1$ ($1 \leqslant i < n$), два различных значения;
 \item Записать в два идущих подряд бита, один из которых ---
  неинициализированный, а значение другого равно \t{x}, два различных
  значения, не равных \t{x}.
\end {itemize}
К примеру, для $n = 4$ возможная процедура инициализации такова:
$\t{uuuu} \rightarrow \t{uuab} \rightarrow \t{ucbb} \rightarrow \t{babb}$;
здесь символом \t{u} обозначаются неинициализированные биты.

Напишите программу, которая:
\begin {itemize}
 \item читает из входного файла последовательность значений, которой надо
  инициализировать память,
 \item проверяет, что такая инициализация возможна и
 \item выводит ответ в выходной файл.
\end {itemize}

\InputFile

Во входном файле содержится один или несколько наборов входных данных.
В первой строке находится одно целое число $T$ ($1 \leq T \leq 10$)
--- количество наборов. Каждый набор описывается двумя строками. Первая из
них содержит целое число $l_i$ ($1 \leq l_i \leq 100 \, 000$) ---
размер памяти ТБК. Вторая строка содержит $l_i$ символов, каждый из которых
равен \t{a}, \t{b} или \t{c} --- последовательность значений, которыми надо
инициализировать память ТБК.

\OutputFile

Выведите в выходной файл $T$ строк, по одному слову на каждой. В $i$-ой
строке должно стоять слово ``\t{YES}'', если для $i$-го набора входных
данных инициализация возможна, и ``\t{NO}'' в противном случае.

\Examples

\begin{example}%
\exmp{
2
4
aabb
4
aaab
}{
NO
YES
}%
\end{example}

\end{problem}
