\begin{problem}{Эйлеров цикл}{euler.in}{euler.out}{2 секунды}

Требуется найти лексикографически минимальный по своей записи эйлеров цикл в графе.

\InputFile

Во входном файле заданы один или несколько графов. Каждый граф задается списком
из не более чем 1995 ребер. Ребро задается тремя числами --- номерами
вершин, которые оно соединяет (не более 44), а также номером ребра. Описание каждого графа
завершается строкой из двух нулей.

\OutputFile

Для каждого графа выведите последовательность номеров ребер, образующих лексикографически
минимальный эйлеров цикл, или сообщение ``\texttt{Round trip does not exist}'', если
эйлеров цикл не существует. После каждого ответа выводите пустую строку.

\Example

\begin{example}
\exmp{
1 2 1
2 3 2
3 1 6
1 2 5
2 3 3
3 1 4
0 0
1 2 1
2 3 2
1 3 3
2 4 4
0 0
}{
1 2 3 5 4 6
~
Round trip does not exist
~
}%
\end{example}

\end{problem}
