\begin{problem}{Разбиения на слагаемые}{part.in}{part.out}{2 секунды}

Найти $k$-е в антилексикографическом порядке разбиение на слагаемые числа $n$ (разбиения
нумеруются с нуля).

\InputFile

Во входном файле задано не более $5\,000$ наборов входных данных. Каждый набор входных данных
состоит из числа $1\le n\le 100$ и $k$, корректно задающего целый неотрицательный номер разбиения.
Входной файл завершается набором 0 0.

\OutputFile

Для каждого из наборов входных данных выведите на отдельной строке разбиение через пробел.

\Example

\begin{example}
\exmp{
4 1
4 3
0 0
}{
3 1
2 1 1
}%
\end{example}

\end{problem}
