\begin{problem}{Два треугольник}{triangle.in}{triangle.out}{2 секунды}

Найдите площадь объединения двух треугольников.

\InputFile

2 строки по 6 чисел:
$x^1_A, y^1_A, x^1_B, y^1_B, x^1_C, y^1_C$ --- координаты вершин первого треугольника,
затем $x^2_A, y^2_A, x^2_B, y^2_B, x^2_C, y^2_C$ ---
координаты вершин второго. Все координаты целые и не превосходят $1\,000$ по
модулю.

\OutputFile

Выведите площадь объединения треугольников с тремя знаками после десятичной точки.

\Example

\begin{example}
\exmp{
0 0 0 1 1 0
1 0 0 1 0 0
}{
0.500
}%
\end{example}

\end{problem}
