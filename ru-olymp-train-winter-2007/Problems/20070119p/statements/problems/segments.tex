\begin{problem}{Отрезки}{segments.in}{segments.out}{2 секунды}

Найдите, есть ли среди $N$ отрезков два пересекающихся.

\InputFile

В первой строке входного файла
записано число $N$ ($1 \le N \le 125 \, 000$) --- количество отрезков.
Следующие $N$ строк содержат описания отрезков: $(i + 1)$-я
строка содержит координаты концов $i$-й сигареты --- целые числа
$x_1$, $y_1$, $x_2$, $y_2$
($-10 \, 000 \le x_1, y_1, x_2, y_2 \le 10 \, 000$).

\OutputFile

В первой строке выходного файла выведите слово ``\t {YES}'', если 
есть два пересекающихся отрезка. Вторая строка должна содержать
числа $i$ и $j$ --- номера двух пересекающихся отрезков.

Если пересекающихся отрезков нет, выведите в единственной
строке выходного файла ``\t {NO}''.

\Example

\begin{example}
\exmp{
2
0 0 2 2
0 2 2 0
}{
YES
1 2
}%
\exmp{
2
0 0 0 5
5 0 5 10
}{
NO
}%
\end{example}

\end{problem}
